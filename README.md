# kndl-automation

## 큰들 아침 기상곡 자동화 시스템
컴퓨터에 파이썬 설치되어있는지 꼭 확인

필요한 라이브러리를 설치하기 위해 아래 명령 실행
    pip install -r requirements.txt

### How
1. selenium.webdriver 사용
2.  kndl.kr에서 유튜브 링크 상위 세 개 가져옴
3.  세 개 실행
